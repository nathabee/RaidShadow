# How to run?
### STEPS 01 :

Clone the repository

```bash
git clone https://github.com/nathabee/RaidShadow
```
### STEP 02- Create a Expo environment 

```bash
cd RaidShadow
npx install
npx  expo start
```

  
### STEP 03- Modify the app
```bash
code .
modify 
```


### STEP 04- Test the app

